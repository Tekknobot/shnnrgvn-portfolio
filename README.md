
# shnnrgvn — Minimal Portfolio (Home + About)

Dark, centered, mobile-first portfolio with only **Home** and **About**.
- No Links page
- No Resume link
- No Contact CTA

## Edit content
- `src/data/profile.json` – name, headline, summary, skills, socials

## Run
npm install
npm run dev

## Build & preview
npm run build
npm run preview
